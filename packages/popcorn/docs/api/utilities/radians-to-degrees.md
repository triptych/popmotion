# `radiansToDegrees`

`radiansToDegrees` converts radian units to degrees.

<TOC />

## Import

```javascript
import { radiansToDegrees } from '@popmotion/popcorn';
```

## Usage

```javascript
radiansToDegrees(0.7); // 40.107...
```

## Types

```typescript
radiansToDegrees(radians: number): number
```
